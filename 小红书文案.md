## 文案4：教程版

**标题：** 手把手教你搭建AI论文阅读系统（免费开源）

---

先看效果 ↓

✨ 一键保存论文
✨ AI自动生成笔记
✨ 数学公式完美渲染
✨ 架构图自动生成
✨ 代码仓库分析

今天教大家怎么自己部署一套！

---

**你需要准备：**

1. 一台服务器（或本地运行也行）
2. Turso账号（免费数据库）
3. AWS S3（存PDF和笔记）
4. Gemini CLI（AI阅读核心）

---

**部署步骤：**

**Step 1: 克隆项目**
```
git clone https://github.com/CurryTang/auto-researcher.git
```

**Step 2: 配置后端**
```
cd backend
npm install
cp .env.example .env
# 填入你的密钥
npm start
```

**Step 3: 配置前端**
```
cd frontend
npm install
npm run dev
```

**Step 4: 安装Chrome插件**
打开 chrome://extensions/
加载 extension/ 文件夹

---

**核心配置项：**

- `TURSO_DATABASE_URL` - 数据库地址
- `AWS_S3_BUCKET` - S3存储桶
- `READER_MAX_PER_HOUR=5` - 每小时处理论文数

---

**技术栈：**

前端：React + Vite + KaTeX + Mermaid
后端：Node.js + Express + Turso
AI：Gemini CLI（主力）+ Claude/GPT（备用）
